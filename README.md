# natural_object_detection

In this repo, I tried to make a Convolution Neural Network, which is capable to classify objects as: <br />
-Airplane <br />
-Car <br />
-Cat <br />
-Dog <br />
-Flower <br />
-Fruit <br />
-MotorBike <br />
-A Person <br />

I used [this](https://www.kaggle.com/prasunroy/natural-images) dataset
